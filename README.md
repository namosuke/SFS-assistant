# SFSアシスタント ver.1.0.0

SFC-SFSを使いやすくするChrome拡張機能です。

## 機能詳細

- 未提出の課題一覧
	アドレスバー横のアイコンをクリックすると、未提出の課題一覧を表示します。

- SFC-SFSロゴからMy時間割に移動

- 内部リンクを新しいタブで開かない

- レポートタイトルの自動入力

- レポート入力画面で課題内容を表示

- レポート文字数カウント

- レポート入力リセットボタン抹消

- レポート提出ボタンの拡大

## 導入方法

1. /srcフォルダを保存
1. Chrome系ブラウザで「ツール」→「拡張機能」を開く
1. 「デベロッパーモード」をオン
1. 「パッケージ化されていない拡張機能を読み込む」からsrcフォルダを選択する
